¿Qué es HTML 5?

HTML es un lenguaje es lenguaje de marcado para la elaboración de páginas web, no un lenguaje de programación

HTML5 es una serie de estándares para diseñar páginas web. Es decir, sirve para controlar la forma en que los navegadores de internet muestran la información y determina la interacción entre el cliente y los contenidos publicados.

HTML5 permite integrar fácilmente el código base de las páginas web y los contenidos multimedia y las apps gracias a su relación directa con CSS y JS.

Así, pues, HTML5, es mucho más versátil y potente que las versiones anteriores.
